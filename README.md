# Kiro Power — Senior Frontend Architecture

Kiro Power com o guia completo da arquitetura frontend Angular da Senior Sistemas.

## Instalação

No Kiro:
1. Abra o painel de Powers (ícone na sidebar)
2. Clique em "Add Custom Power"
3. Selecione "Git Repository"
4. Cole a URL do repositório
5. Clique em "Add"

## O que faz

Ensina o Kiro a:

- Entender a arquitetura de projetos frontend Angular (pastas, padrões, convenções)
- Criar novos projetos do zero seguindo o padrão `<dominio>-<servico>-frontend`
- Criar features CRUD com o padrão de componentes, services, configs e traduções
- Usar corretamente a biblioteca `@seniorsistemas/components-ai`
- Seguir o padrão de traduções i18n com chaves flat `<dominio>.<servico>.<chave>`
